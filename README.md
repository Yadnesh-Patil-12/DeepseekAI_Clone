# 🤖 DeepseekAI Clone - Full Stack AI Chat App

## 🚀 Overview

DeepseekAI Clone is a full-stack AI-powered chatbot web application inspired by modern AI tools like ChatGPT and DeepSeek.
It allows users to interact with AI and receive intelligent responses in real-time.

This project demonstrates full-stack development using the MERN stack with AI API integration.

---

## 🛠️ Tech Stack

### 💻 Frontend

* React.js
* Tailwind CSS

### 🌐 Backend

* Node.js
* Express.js

### 🗄️ Database

* MongoDB

### 🔐 Authentication

* JSON Web Tokens (JWT)

### 🤖 AI Integration

* OpenAI API / DeepSeek API

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/Yadnesh-Patil-12/DeepseekAI_Clone.git
cd DeepseekAI_Clone
```

---

### 2️⃣ Install Dependencies

#### Frontend

```
cd client
npm install
```

#### Backend

```
cd ../server
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env` file inside the **server** folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_api_key
```

---

## ▶️ Run the Project

### Start Backend

```
cd server
npm run dev
```

### Start Frontend

```
cd client
npm start
```
